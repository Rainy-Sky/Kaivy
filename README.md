# 愤怒的小鸟
## 操作指南
### 主界面
  点击 音响图标 控制音乐的开关
  点击 EXTRA 图标进入游戏规则说明
  点击 左下角图标 进入游戏难度选择
  点击 中间 PLATY 图标开始游戏
### 游戏难度选择
  选择 简单 难度模式，游戏中可使用小鸟数量增加，改为3个红色小鸟 两个黄色小鸟
  选择 中等 难度模式，游戏中小鸟为3个红色小鸟 一个黄色小鸟
  选择 困难 难度模式，游戏中小鸟数量与中等难度相同，但小猪的血量略微上调
### 游戏规则
  点击 中间PLAY 进入游戏
  在游戏进行界面 在弹弓附近按住小鸟，拉开一定距离即可发射小鸟，小鸟会在飞行途中与木栅或小猪碰撞
  小鸟与木栅和小猪的碰撞过程中如果计算的物理冲力超过一定阈值，即可“打破”木栅和小猪
  木栅和小猪被“打破”之后可获得相应积分
  若关卡中小猪被全部击败，即可游戏通关，并根据相应的积分给与关卡星级，反之则游戏失败
### 游戏界面
  点击 音响图标 控制音乐的开关
  点击左上角暂停进入暂停，可以在暂停界面重新开始、退出游戏或继续游戏
  关卡通关后可在通关界面选择重新开始或进入下一关
  关卡失败后进入失败界面选择重新开始
### 特殊规则
  |按键|功能|
  |**S**|**进入无重力模式**|
  |**N**|**恢复正常模式**|
  |**W**|**开启游戏边界**|
  |**B**|**进入逆风模式**|
  |**SPACE**|**使用小鸟能力**|
  ######  黄色小鸟在飞行中按　SPACE　可进行加速
